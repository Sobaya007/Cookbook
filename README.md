# 使い方
## リポジトリ構成
- docs
    - `*.html` 生成ドキュメント 
- source
    - `*.d` 目的別のサンプルソース

### 補足
- 誤字などなく常に実行できることを保証するため、HTMLはコードから生成する構造としています。
- dubを使って単体テストが通るようにしてあります。

## 実行方法

```bash
dub test
```

## ドキュメントとして出力する方法

```bash
dmd build --build=docs
```
