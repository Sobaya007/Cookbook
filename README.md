# 使い方
## リポジトリ構成
- docs
    - 目的別フォルダ
        - `app.d` 
        - `app.html` 生成したHTML

### 補足
- 目的別フォルダは1つのプロジェクトとして構成しています。
- 誤字などなく常に実行できることを保証するため、HTMLはコードから生成する構造としています。

## 実行方法

```bash
cd ./docs/string
dmd -unittest -main -run app.d
```

## ドキュメントとして出力する方法

```bash
cd ./docs/string
dmd -D -o- app.d
```
